# vim-conf
simple vim conf

# Introduction

1. Creating ```~/vimrc/``` 
```bash
mkdir ~/vimrc
cd ~/vimrc
git init
```

2. Creating vim config file in ```~/vimrc/```   
you can copy config on ```~/vimrc/``` if you have ```.vimrc``` or ```.vim/``` 

```bash
# cd ~/vimrc
touch vimrc
mkdir vim
```

3. link file to `~/.vimrc` and `~/.vim`

```bash
ln -s ~/vimrc/vimrc ~/.vimrc
ln -s ~/vimrc/vim ~/.vim
```
