# vimrc
simple vim conf

# Introduction

1. Create ```~/vimrc/``` 
```bash
mkdir ~/vimrc
cd ~/vimrc
git init
```

2. Create vim config file in ```~/vimrc/```   
```bash
cd ~/vimrc
touch vimrc
mkdir vim
```

3. Create Symbolic link to `~/.vimrc` and `~/.vim`
```bash
ln -s ~/vimrc/vimrc ~/.vimrc
ln -s ~/vimrc/vim ~/.vim
```
